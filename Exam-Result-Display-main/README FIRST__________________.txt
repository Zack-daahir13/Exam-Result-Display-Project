## üõ†Ô∏è Getting Started

### 1. Install XAMPP
To run this project, you need to have a local server environment set up. We recommend using **XAMPP**:
- Download XAMPP from [here](https://www.apachefriends.org/index.html).
- Install it on your machine and start the Apache and MySQL modules.

### 2. Set Up the Project
- **Clone the repository** or **download the project files**.
- Copy the project folder into the `htdocs` directory located in the XAMPP installation folder (e.g., `C:\xampp\htdocs\`).

### 3. Configure the Database
- Open **phpMyAdmin** by navigating to `http://localhost/phpmyadmin/` in your web browser.
- Create a new database (e.g., `exam_results`).
- Import the provided SQL file (`exam_results.sql`) into the newly created database.

### 4. Update the Configuration
- Open the project folder and locate the `config.php` file.
- Update the database connection details (`hostname`, `username`, `password`, `database`) to match your local setup.

### 5. Run the Project
- Navigate to `http://localhost/exam-result-display/` in your web browser.
- You should see the homepage of the **Exam Result Display** application.
